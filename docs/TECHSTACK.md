## 1. Overview

This project uses a modern full-stack JavaScript ecosystem focused on performance, scalability, and rapid development. The chosen technologies align with industry standards and are optimized for building responsive, component-based web applications.

---

## 2. Core Tech Stack

### Frontend Framework

**Next.js (App Router)**

**Justification:**

* Provides built-in routing and server-side rendering (SSR)
* Optimized performance with static generation support
* Seamless deployment on Vercel
* Supports API routes for backend functionality

**Usage:**

* Page routing via `/app` directory
* Layout management
* API endpoints for form handling

---

### Library

**React (Functional Components + Hooks)**

**Justification:**

* Component-based architecture improves reusability
* Hooks simplify state and lifecycle management
* Widely adopted industry standard

**Usage:**

* Building modular UI components (Navbar, Hero, etc.)
* Managing state (form inputs, UI interactions)

---

### Styling Framework

**Tailwind CSS**

**Justification:**

* Utility-first approach enables rapid development
* Ensures consistent spacing, typography, and layout
* Reduces need for custom CSS
* Excellent support for responsive design

**Usage:**

* Styling all UI components
* Implementing responsive layouts
* Maintaining design consistency

---

### API Layer

**Next.js API Routes**

**Justification:**

* No need for separate backend setup
* Lightweight and easy to implement
* Suitable for mock data handling and simple integrations

**Usage:**

* Handling lead capture form submissions
* Returning mock responses

---

### Deployment

**Vercel**

**Justification:**

* Native support for Next.js projects
* Fast global CDN
* Automatic deployments from GitHub
* Zero configuration required

**Usage:**

* Hosting the live application
* Continuous deployment

---

### Version Control

**Git + GitHub**

**Justification:**

* Tracks changes and maintains version history
* Enables collaboration and code management
* Required for deployment workflows

**Usage:**

* Source code management
* Repository submission
